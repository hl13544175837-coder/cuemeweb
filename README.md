# CuemeWeb

> 夸克自研 AI 智能助手，让每个人都能轻松产出专业级长文内容

## 核心能力

### 超长文本生成
- 支持最长 **2 万字**内容生成，远超同类 AI 写作工具
- 大纲 → 正文分步创作，确保逻辑连贯不跑题

### 千种体裁覆盖
- 预训练覆盖 1000+ 种细分体裁：学术论文、商业报告、小红书文案、小说、诗歌等
- 支持文风定制，可模仿特定作者或平台调性

### 小伙伴广场
- AI 虚拟角色互动平台，将工具型 AI 转化为情感化陪伴型 AI
- 支持 IP 联动角色（如《百炼成神》）、自定义角色创建
